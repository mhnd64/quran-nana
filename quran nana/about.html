Project Overview & Architecture

Learn about the database structures, styling systems, and interactive modules of the portal.

1. Academic Requirement Scope

This project is designed to combine robust back-end server mechanics (PHP & MySQL) with separate, clean static front-end assets to meet advanced university requirements:

  • HTML5 Structure: Semantic tags, LTR layout alignment, descriptive title tags, and clean class references.
  • CSS3 Styling: Centralized properties, variables (`:root`), flexbox alignments, multi-column grid systems, and glassmorphic gradients.
  • JavaScript Actions: Client-side real-time query filtering, automated notification dismissals, and dynamic navigation badges.

2. Integrated Database Schema

The SQL database backend runs on a highly normalized relational model featuring three integrated tables:

  • `user`: Manages credentials, hashed passwords (`password_hash`), and authentication roles (`admin` or `student`).
  • `quran_verses`: Stores full Quranic texts with indexed attributes supporting fast substring matching.
  • `scientific_miracles`: Holds cosmic research papers, linked directly to the verses table via composite foreign fields.
